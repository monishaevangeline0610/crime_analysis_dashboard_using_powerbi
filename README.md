# Crime Analysis Dashboard (Power BI)

Interactive Power BI dashboard analyzing crime data across selected Indian states (2014–2025). Includes the .pbix dashboard file, dataset (CSV), and preview images.

Tools
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)

Project structure
```
crime_analysis_dashboard_using_powerbi/
├─ Dashboard/India Crime Analysis.pbix
├─ Dataset/Crime Dataset.csv
├─ images/dashboard1.png
└─ README.md
```
